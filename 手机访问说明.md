# 手机访问说明

这个系统运行在你的电脑上，手机要访问，需要满足两个条件：

1. 电脑和手机连接同一个 WiFi。
2. 用 `04-start-mobile.cmd` 启动系统。

## 使用步骤

1. 在电脑上双击 `04-start-mobile.cmd`
2. 黑色窗口会显示类似这样的地址：

```text
http://192.168.1.8:3000
```

3. 在手机浏览器里输入这个地址。
4. 如果 Windows 弹出防火墙提示，请点允许访问。

## 如果手机打不开

请检查：

- 手机和电脑是不是同一个 WiFi。
- 电脑黑色窗口有没有关掉，关掉后系统就停了。
- Windows 防火墙有没有允许 Node.js 访问专用网络。
- 地址里不能写 `localhost`，手机上必须写电脑的 IP，例如 `http://192.168.1.8:3000`。
