# 房产销冠客户管家：怎么打开

这个项目不是双击某个 `.md` 文件打开的软件，它是一个本地 Web 系统。

你需要先启动服务，然后用浏览器访问：

```text
http://localhost:3000
```

## 最简单打开方式

在当前文件夹里，按顺序双击：

1. `01-install-dependencies.cmd`
2. `02-init-database.cmd`
3. `03-start-app.cmd`

第三步启动后，不要关闭黑色命令窗口。

然后打开浏览器访问：

```text
http://localhost:3000
```

## 那两个 md 文件有什么用？

`.md` 文件是说明文档或提示词文件，不是软件入口。

- `README.md`：给开发者看的项目说明。
- `prompts/customer-analysis.md`：给 AI 分析客户时用的提示词模板。
- `使用说明-先看这个.md`：给你看的启动说明。

真正的软件页面在浏览器里打开：

```text
http://localhost:3000
```

请把你双击 `.cmd` 后窗口里出现的错误内容发给我。
