# 手机公网临时链接说明

如果手机无法通过 `192.168.x.x:3000` 访问，通常是路由器或网络做了设备隔离。

可以改用临时公网链接：

1. 双击 `07-start-public-link.cmd`
2. 它会打开两个黑色窗口。
3. 等待窗口里出现类似：

```text
your url is: https://xxxx.loca.lt
```

4. 在手机浏览器打开这个 `https://xxxx.loca.lt` 地址。

注意：

- 两个黑色窗口都不要关闭。
- 这是临时链接，关闭窗口后就失效。
- 第一次运行可能会下载 `localtunnel`，需要电脑能联网。
- 如果页面要求输入 endpoint IP password，按页面提示输入电脑公网 IP。
